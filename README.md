# Menu

1.自定义菜单内容

2.依据上述数据创建一个menu

3.把上述菜单添加到app身上

**label**字段是显示的文字，**type**字段有[radio,checkbox,submenu,separator]这几种类型，**accelerator**字段是快捷键，**submenu**表示子列表，还有**icon**字段，当然还可以在menu的列表写**click()**回调函数
